# Tugas Visualisasi Data - Dataset Tips

Tugas ini berisi program Python untuk memvisualisasikan data dari file `tips.csv`. 
1. **Scatter Plot**: Menampilkan hubungan antara hari (day) dan jumlah tip.
2. **Pie Chart**: Menampilkan persentase jumlah pengunjung laki-laki dan perempuan.
3. **Bar Chart**: Menampilkan jumlah pengunjung berdasarkan hari.
